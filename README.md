## What I Learned

This project helped me improve my understanding of:

- Using `max-width` for responsive components
- Applying `box-sizing: border-box` globally for predictable layout behaviour
- Mobile-first responsive design
- Using Flexbox to centre and structure layouts
- Converting pixel-based designs into `rem` for scalability

Example of using global styling:

```css
*,
*::before,
*::after {
    box-sizing: border-box;
}
```
